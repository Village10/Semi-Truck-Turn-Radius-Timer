# Semi-Truck-Turn-Radius-Timer

## Usage
1. Install the timer using the steps below
2. Run the .exe
3. Open [the game](https://drivingskillsforlife.com/media/games/vr2pc/sub/semitruck-turn_radius/index.html)
4. Make sure you are in fullscreen
5. Press shift to stop the timer
6. Press any key to start the timer
7. The timer will turn red if you had not complete the level when you pressed shift, so make sure to only press it when you see the finish screen
8. Any times under 3 seconds are not counted

## Installation
1. Go to the [Releases Page](https://github.com/Village10/Semi-Truck-Turn-Radius-Timer/releases)
2. Download the SemiTruckTimer.exe attatched to the latest release
